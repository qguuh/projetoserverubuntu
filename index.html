Sobre o Ubuntu Server

O Ubuntu Server é um sistema operacional baseado em Linux amplamente utilizado em servidores pela sua estabilidade, segurança e facilidade de administração. Durante o curso, aprendemos a instalar e configurar o Ubuntu Server em uma máquina virtual, compreender sua estrutura de diretórios, trabalhar com serviços essenciais e realizar tarefas administrativas básicas.
Entre os conteúdos estudados, estão:

  • Criação e gerenciamento de usuários
  • Configuração de serviços web
  • Utilização do Git
  • Rotina de administração

Esse aprendizado forma a base para administrar servidores Linux em ambiente profissional, garantindo que aplicações e páginas web funcionem de forma estável, organizada e segura.
